Design of ad-hoc network infrastructure in Linux
Nov 2019 - Dec 2019

• Designed ad-hoc network architecture by configuring DNS and DHCP server for IPV4 and IPV6 allocation
• Constructed apache web server for storing and sharing, deployed backup server through Secure Shell and cron tab for regular backups, added Firewall for security by allowing and denying specific requests
• Tested network security by ARP poisoning, implemented IPSec tunnels for secure communication between servers and NFS for mounting shared directories

Design and simulation of inter-networking for a multi-national company using Cisco Packet Tracer
Oct 2019 - Nov 2019

• Designed a network hierarchy for a small-scale stable company that connects their main and sub-offices worldwide
• Used VLAN for main and sub-offices for easier administration and security, performed optimal allocation of IP addresses through DHCP servers
• Implemented OSPF as the routing protocol and Frame-Relay to improve the data rate
• Incorporated STP and HSRP for loop free topology and fault tolerance and applied Access Control Lists to enhance security
